# graphqlDemo

## Discription

Sample NodeJS server with ES6, Typescript and Graphql

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
